\section{收集数据}\label{sec:Notation}

\textbf{参数获取方法}：通过实地测试获取加速度$a_1$和减速度$a_2$，测试条件包括：
\begin{itemize}[leftmargin=2em]
    \item 使用普通家用轿车在平整沥青路面进行
    \item 由经验司机完成加速/减速操作
    \item 副驾驶助手记录速度-时间对应数据
    \item 测试重复3次取平均值
\end{itemize}
